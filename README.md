# Ядро DSL для задач семейства RCPSP: примитивы, грамматика, семантика
**RCPSP** - это задача планирования проектов с ограниченными ресурсами и одна из ключевых задач в управлении проектами, цель которой - составить оптимальное расписание работ проекта так, чтобы минимизировать его общую продолжительность, при этом учитывая зависимости между задачами и ограниченное количество доступных ресурсов. Существует огромное количество постановок подобных задач, отличающихся по входным данным, метрикам оптимизации и структуре.

Данный проект использует **DSL (Domain-Specific Language)** на базе JSON для описания задач планирования. Архитектура построена на принципе композиции: базовая структура (ядро) описывает универсальный граф расписания, а специфичные предметные ограничения инкапсулируются в блоке `extensions`

Текущая реализация поддерживает два расширения:
1. RCPSP (Resource-Constrained Project Scheduling Problem) — классическое проектное планирование.
2. Cluster — планирование вычислительных задач (например, обучение ML-моделей) на серверах и GPU-кластерах
